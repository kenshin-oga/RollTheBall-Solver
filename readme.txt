「roll the ball」というパズルゲームを探索アルゴリズムを用いて解くSolverを制作した。
環境は、VisualStudio2019、DXライブラリ、C言語を用いた。
　また、反復深化法とIDASTARという２つの探索アルゴリズムを用いて、３つのSolverをつくった。

  はじめに用いる探索法とステージを決める。その次の画面で最短手数いくつの問題を解くか
選択し、次の画面でEnterを押すと探索を始める。終了するとコンピュータ内の処理時間とノ
ード数が表示され、ソリューションファイルと同じ場所に「Movelog.txt」が生成される。こ
こには、正解最短手数と具体的な手順が全通り記入される。また、最初の画面でqを押すと
パズルスライドの様子を描画するか選べる画面に進める。「する」にした場合処理時間が
大幅に増えるので注意してほしい。
　実装した中で最も優秀な探索アルゴリズムはIDA*の「クリア盤面と違うパネルの数」を
ヒューリスティック関数として採用したもので、15手問題まで安定して解くことができた。
